Tìm Lịch Rảnh

⚠️ Cần setup Firebase

Bạn chưa cấu hình Firebase! Vui lòng:

  1. Tạo Firebase project tại console.firebase.google.com
  2. Tạo Realtime Database (test mode)
  3. Lấy Firebase Config từ Project Settings
  4. Paste config vào dòng 433 của file HTML này
1. Chọn khoảng thời gian
2. Quản lý người tham gia
Chưa có ai. Thêm người tham gia để bắt đầu
3. Chọn tên của bạn để điền lịch
4. Điền lịch rảnh (click vào ô)
💡 Cách dùng: Click vào các ô để đánh dấu bạn rảnh. Ô có chấm đen góc trái = bạn đã tick. Màu xanh đậm = nhiều người rảnh. Hover vào ô để xem chi tiết. Tự động lưu mỗi lần bạn click.
Kết quả: Thời gian tất cả đều rảnh
